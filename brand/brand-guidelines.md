# Brand Guidelines

BS Engineering visual identity is designed to communicate clarity, credibility and technical professionalism.

## Style direction

- Minimal
- Technical
- Clean
- Structured
- Calm
- Evidence-based

## Color palette

| Token | Hex | Purpose |
|---|---|---|
| Primary | `#0F172A` | Main text and dark surfaces |
| Accent | `#2563EB` | Links, highlights and key accents |
| Secondary | `#475569` | Secondary text |
| Muted | `#64748B` | Supporting text |
| Background | `#FFFFFF` | Main document background |
| Soft background | `#F8FAFC` | Cards and subtle sections |
| Line | `#E2E8F0` | Dividers and borders |

## Typography

Primary resume typography: Roboto.

Design system / portfolio typography: Inter or Roboto depending on medium.

## Design principles

- Use white space intentionally.
- Avoid decorative visuals that do not support meaning.
- Use icons sparingly.
- Keep contrast high.
- Use consistent section hierarchy.
- Make documents readable before making them visually impressive.

## Document principle

The design should never hide weak content. The role of design is to make strong, credible content easier to understand.
