# LinkedIn 2.0

This directory stores the LinkedIn profile strategy and reusable text blocks.

## Profile direction

LinkedIn should support two main pillars:

1. Quality / Metrology as the current professional foundation
2. Manufacturing Systems as the strategic development direction

## Recommended headline

Engineering Systems Specialist | Quality & Metrology | Manufacturing Systems | Data Analysis | Linux & Web Technologies

## Profile story

The profile should communicate a coherent story:

- practical experience in manufacturing and quality,
- dimensional metrology and technical documentation,
- IT/Linux/web background,
- interest in systems, data, automation and AI-assisted engineering,
- credible continuous learning.

## Update order

1. Headline
2. About
3. Experience
4. Skills
5. Featured section
6. Projects
7. Banner
