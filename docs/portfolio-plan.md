# Portfolio Plan

The portfolio should grow from a few carefully selected projects, not from a long list of unfinished ideas.

## Priority order

1. Manufacturing ERP Platform
2. HydrogenHTTPD
3. Android GIS Application
4. AI Engineering Lab
5. Quality Tools

## Standard for each project

Each project should eventually include:

- problem statement,
- purpose,
- architecture,
- technology choices,
- screenshots or diagrams,
- current limitations,
- next steps.

## Goal

Show technical thinking in a way that feels practical, structured and credible.
