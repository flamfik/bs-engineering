# BS Engineering

**Engineering better systems through data, quality and technology.**

BS Engineering is the professional engineering brand and documentation hub of Bartosz Suchy. It connects practical experience in manufacturing quality, dimensional metrology, IT systems, Linux administration, web technologies, documentation, data analysis and continuous learning.

This repository is not a traditional software project. It is a structured professional knowledge base used to document career development, engineering principles, portfolio projects, resume versions, LinkedIn content and learning roadmap.

## Core direction

BS Engineering focuses on the intersection of:

- Manufacturing quality
- Industrial metrology
- Technical documentation
- IT systems and Linux administration
- Web technologies and databases
- Data and process analysis
- AI-assisted engineering workflows
